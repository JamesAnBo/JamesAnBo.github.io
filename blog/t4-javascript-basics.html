Javascript Basics

  • Things I've learned about Javascript.

  • How does JavaScript compare to HTML and CSS?

  • While HTML and CSS pertain to how a website looks and feels. Javascropt allows for the website to be more interactive. It gives an object far more complex rules and instructions to follow.

  • Explain control flow and loops using an example process from everyday life, for example 'waking up' or 'brushing your teeth'.

  • A control flow is just like an if/else statement in normal life. For example you might think that if it's sunny i'll mow the lawn, otherwise i'll stay inside. A loop is exactly what it sounds like. In everyday life you might think I'll keep chopping this onion until it's all chopped up. Thus you're chopping it on a loop until you've met a requirement.

  • Explain the difference between accessing data from arrays and object literals.

  • An array is like a list with a specific order. To access data from an array you need to know where in the array the data is or what the data you're accessing is (Like a list of names or dates). When accessing data from object literals you're essentially access descriptive attributes of an object (Like how tall someone is, or what breed a dog is).

  • Explain what functions are and why they are useful.

  • A function in Javascript is a named block of code that you might want to run several times and on demand. Often with different variables in place. Such as x+y, but you want a different number for x and y each time you call it. It's useful to keep you're code to a minimum. For example it's easier to write

    
function exampleFunc(x,y){
  if(x > 5) {
    return x + y;
  }else{
    return x - y;
  }
}

    and follow that up with

    
exampleFunc(3,5);
exampleFunc(7,2);
exampleFunc(243,32);

    than rewrite the same if/else over and over again. Also it saves time and code length if you wish to call the "exampleFunc" function inside another function.
