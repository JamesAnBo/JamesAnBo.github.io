HTML, CSS & DOM Part 2

  • Things I've learned from
    Udacity HTML and CSS lesson 2

  • What is grid based design?

  • A grid based design is when a webpage is divided into rows and columns. It allows for an easy layout to both view and develop with.

  • Whats all the hype about responsive webpages?

  • Responsive webpages adjust to fit the screen its being viewed on. This allows a webpage to shuffle and resize elements to allow for the best viewing experience wether on the desktop or on a smartphone.

  • What is semantic structure?

  • Semantic structure is the use of tags in the HTML code to define different elements of a webpage. It also refers to the orderlyness of the HTML code used. Properly written code is easy to work with and to spot what elements are assigned to what tags.
